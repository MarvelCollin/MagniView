Getting Started

Installation

Simply include the MagniView CSS file in your HTML:

<link rel="stylesheet" href="magniview.css">

Usage

Start using MagniView's utility classes in your HTML:

<div class="flex items-center justify-between p-4 bg-blue-500 text-white">
  <h1 class="text-2xl font-bold">MagniView Demo</h1>
  <button class="bg-white text-blue-500 px-4 py-2 rounded">Click Me</button>
</div>
